# TWRP Device Tree for Samsung Galaxy A54 5G

The Galaxy A54 5G (codenamed _"a54x"_) is an upper-mid-range smartphone from Samsung.

It was announced and released in March 2023.

## How to build

This device tree was tested and is fully compatible with [minimal-manifest-twrp](https://github.com/minimal-manifest-twrp/platform_manifest_twrp_aosp).

1. Set up the build environment following the instructions [here](https://github.com/minimal-manifest-twrp/platform_manifest_twrp_aosp/blob/twrp-12.1/README.md#getting-started)

2. In the root folder of the fetched repo, clone the device tree:

```bash
git clone https://github.com/Vaz15k/android_device_samsung_a54x.git -b twrp-12.1 device/samsung/a54x
```

3. To build:

```bash
export ALLOW_MISSING_DEPENDENCIES=true
. build/envsetup.sh
lunch twrp_a54x-eng
mka recoveryimage
```
